




# Crypto Miner Clicker

A small, fun idle/clicker game about mining Bitcoin. Built with pure HTML, CSS, and JavaScript.

## Features
- Click to mine BTC
- Buy upgrades that generate passive income
- Prestige system (permanent multiplier)
- Auto-save (localStorage)
- Mobile-friendly

## How to Play
1. Open `index.html` in your browser
2. Click the big coin!
3. Buy miners to earn BTC automatically
4. Prestige when you have enough total mined BTC
